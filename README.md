## Запускаем инит
```
python marketplace/manage.py db init
```
## Запускаем миграцию
```
python marketplace/manage.py db migrate
```
## Применяем обновления
```
python marketplace/manage.py db upgrade
```
## Коннектимся к БД
```
psql -h localhost -p 5432 -U testuser marketplace
```
